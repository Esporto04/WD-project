Web Hosting

Explanation: Web hosting is a service that allows websites to be available on the internet.

All website files like HTML, CSS and images are stored on a server.

HTTP

Explanation: HTTP stands for Hyper Text Transfer Protocol.

It is used to transfer data between browser and server.

Example:

You enter a website → browser sends request → server sends response.
