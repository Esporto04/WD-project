<table> Tag

Explanation: Used to create table.

Example:

<table border="1">
<tr>
<th>Name</th>
<th>Marks</th>
</tr>
<tr>
<td>Shlok</td>
<td>90</td>
</tr>
</table>

Output:

Name Marks
Shlok 90
